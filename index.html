Sobre mim

Sou estudante de Mestrado em Humanidades Digitais na Universidade do Minho, com grande interesse em Análise de Dados, Web Scraping e automatização com Inteligência Artificial. Gosto de aplicar o meu pensamento crítico às ferramentas tecnológicas que uso para compreender e explorar dados de forma prática e criativa.

Tenho experiência com Python e bibliotecas como Pandas e Matplotlib, além de explorar o uso de APIs e ferramentas low-code (Make.com, Zapier, Flowise) para integrar dados e criar sistemas automatizados. O meu objetivo é combinar o olhar humano e analítico das Humanidades com o poder das tecnologias de dados.
