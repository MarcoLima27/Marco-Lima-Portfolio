Criação de Website: Associação Protetora Animal de Ponte (APAP)

Projeto desenvolvido no âmbito do Mestrado em Humanidades Digitais — Unidade Curricular de Criação de Contéudo em Ambientes Virtuais (1º ano)

O objetivo deste projeto foi criar um website completo para a Associação Protetora Animal de Ponte (APAP), que é uma organização sem fins lucrativos, criada por voluntários. O website foca-se na explicação das secções de voluntariado, doações e adoção responsável. O trabalho envolveu a organização da informação de forma clara e acessível, tornando o site intuitivo para diferentes públicos.

O objetivo deste projeto foi criar um website completo para a Associação Protetora Animal de Ponte (APAP), que é uma organização sem fins lucrativos, criada por voluntários. O website foca-se na explicação das secções de voluntariado, doações e adoção responsável. O trabalho envolveu a organização da informação de forma clara e acessível, tornando o site intuitivo para diferentes públicos.

As principais tarefas incluíram:

  • Comunicação próxima seguida de várias análises das necessidades da associação;
  • Criação da estrutura de páginas e navegação eficiente entre vários alunos, de forma a criar um website homogéneo;
  • Desenvolvimento de conteúdos visuais e textuais para explicar a missão da APAP;
  • Implementação de secções de doações, voluntariado e adoção, de forma a tornar a experiência do usuário prática e interativa;
  • Testes de usabilidade para garantir o funcionamento e a responsividade do website.

O resultado foi um website funcional e visualmente apelativo. Este projeto fortaleceu as minhas competências em desenvolvimento web, design responsivo, comunicação digital, gestão de conteúdos e também o trabalho e comunicação em grupo.
