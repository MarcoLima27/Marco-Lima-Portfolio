Web Scraper — Genius.com

Projeto desenvolvido no âmbito da Unidade Curricular de Processamento de Linguagem Natural (Mestrado em Humanidades Digitais).

Este projeto consistiu na criação de um web scraper em Python desenvolvido para recolher letras de músicas diretamente do website Genius.com. O objetivo foi explorar técnicas de extração e estruturação de dados textuais de forma a aplicar boas práticas de scraping e tratamento de dados Este projeto foi desenvolvido em duplas, o que também se verificou benéfico em termos de aprendizagem de trabalho em equipa e estruturação de código em simultâneo. Foi bastante gratificante aprender a programar em conjunto e a adaptar o código para encaixar de forma precisa, seguida de vários testes de funcionalidade em conjunto.

O script permite recolher, para cada música, as seguintes informações:

  • Título da música
  • Autor / Artista
  • Data de criação ou lançamento
  • Álbum
  • Letra completa

Após a recolha, os dados são automaticamente exportados para um ficheiro CSV e outro em JSON, de forma a facilitar análises posteriores em Python (com Pandas) ou ferramentas de visualização.

O projeto melhorou as minhas competências em web scraping, limpeza de dados, manipulação de ficheiros e automação de fluxos de trabalho. Foram utilizados módulos como Requests, BeautifulSoup e Pandas para estruturar o código e otimizar a extração.

Este trabalho consolidou a minha compreensão sobre a extração de dados textuais em larga escala e sobre o papel do Processamento de Linguagem Natural no tratamento de textos obtidos a partir da web.
